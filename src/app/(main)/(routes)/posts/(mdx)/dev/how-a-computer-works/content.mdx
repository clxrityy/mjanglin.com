<CPUHeading />

If you're not familiar with the term, a CPU (Central Processing Unit) is often referred to as the "brain" of the computer.

With it's small and silicon chip, it performs the majority of the processing tasks in a computer.
- This chip contains billions of microscopic switches called [transistors](https://newsroom.intel.com/tech101/the-transistor-explained) that can be turned on or off to represent binary data (0s and 1s).
<Transistor />
    - The value of a transistor is called a [bit](https://web.stanford.edu/class/cs101/bits-bytes.html).
    - A group of 8 bits forms a [byte](https://web.stanford.edu/class/cs101/bits-bytes.html), which is the basic unit of data in computing.
        - This is the smallest addressable unit of memory in a computer, which is 256 different combinations of 0s and 1s.

CPUs, with their delicate and intricate designs, are one of the most complex components in a computer. It takes such precision to manufacture them that a single CPU can contain billions of transistors, all working together to perform calculations and execute instructions at incredible speeds.
If you're interested in seeing just how these things are made, check out this [video](https://www.youtube.com/watch?v=2ehSCWoaOqQ) where someone takes a tour of an Intel factory.

<BinaryHeading />

These variations of 0s and 1s are used to represent all types of data in a computer, including numbers, letters, images, and sounds.
- For example, the letter "`A`" is represented by the binary number `01000001`, which is 65 in decimal.
- This binary representation is used by the CPU to perform calculations and execute instructions.

Binary is complex and can be difficult to interpret (for a human). Reading 0s and 1s is not intuitive, so we use a system called [hexadecimal](https://en.wikipedia.org/wiki/Hexadecimal) to make it easier.
- A group of 4 bits can be represented by a single hexadecimal digit, which ranges from 0 to F (0-15 in decimal).
<Hexidecimal />
    
- This allows us to represent large binary numbers in a more compact and readable format.
